# List Teams

Get a list of teams that the authenticated user has access to.

## Endpoint

```http
GET ${baseUrl}/teams
```

## Query Parameters

| Parameter | Type   | Description                             |
| --------- | ------ | --------------------------------------- |
| page      | number | Page number for pagination (default: 1) |
| limit     | number | Number of teams per page (default: 10)  |

## Response

```typescript
interface TeamListResponse {
  success: boolean
  data: {
    team: {
      id: number
      name: string
    }
    members: {
      id: number
      role: string
      status: string
      email: string
    }[]
  }[]
}
```

## Example Request

```typescript
const response = await fetch('${baseUrl}/teams', {
  headers: {
    Authorization: 'Basic YOUR_API_KEY',
  },
})

const data = await response.json()
```

## Error Codes

| Status Code | Description  |
| ----------- | ------------ |
| 200         | Success      |
| 401         | Unauthorized |
| 500         | Server Error |
