# Update Team

Update an existing team's settings.

## Endpoint

```http
PUT ${baseUrl}/teams/{teamId}
```

## Path Parameters

| Parameter | Type   | Description              |
| --------- | ------ | ------------------------ |
| teamId    | number | The ID of team to update |

## Request Body

```typescript
interface UpdateTeamRequest {
  name?: string
  settings?: {
    allowMemberInvites?: boolean
    defaultMemberRole?: 'MANAGER' | 'USER' | 'SUPERVISOR'
    fileAccessLevel?: 'TEAM' | 'INDIVIDUAL'
  }
}
```

## Response

```typescript
interface UpdateTeamResponse {
  success: boolean
  data: {
    id: number
    name: string
    settings: {
      allowMemberInvites: boolean
      defaultMemberRole: string
      fileAccessLevel: string
    }
    updatedAt: string
  }
}
```

## Example Request

```typescript
const response = await fetch('${baseUrl}/teams/123', {
  method: 'PUT',
  headers: {
    'Content-Type': 'application/json',
    Authorization: 'Basic YOUR_API_KEY',
  },
  body: JSON.stringify({
    name: 'Updated Team Name',
    settings: {
      allowMemberInvites: true,
      fileAccessLevel: 'TEAM',
    },
  }),
})

const data = await response.json()
```

## Error Codes

| Status Code | Description               |
| ----------- | ------------------------- |
| 200         | Team Updated Successfully |
| 400         | Invalid Request Body      |
| 401         | Unauthorized              |
| 403         | Forbidden                 |
| 404         | Team Not Found            |
| 500         | Server Error              |
