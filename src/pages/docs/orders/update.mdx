# Update Order

Update an existing order's details.

## Endpoint

```http
PUT ${baseUrl}/orders/{orderId}
```

## Path Parameters

| Parameter | Type   | Description               |
| --------- | ------ | ------------------------- |
| orderId   | number | The ID of order to update |

## Request Body

```typescript
interface UpdateOrderRequest {
  priority?: number
  instructions?: string
  status?: OrderStatus
  options?: {
    customFormatting?: boolean
    timestamping?: boolean
    verbatim?: boolean
    rushDelivery?: boolean
  }
}
```

## Response

```typescript
interface UpdateOrderResponse {
  success: boolean
  data: {
    id: number
    fileId: string
    status: OrderStatus
    priority: number
    instructions?: string
    options: {
      customFormatting: boolean
      timestamping: boolean
      verbatim: boolean
      rushDelivery: boolean
    }
    updatedAt: string
  }
}
```

## Example Request

```typescript
const response = await fetch('${baseUrl}/orders/123', {
  method: 'PUT',
  headers: {
    'Content-Type': 'application/json',
    Authorization: 'Basic YOUR_API_KEY',
  },
  body: JSON.stringify({
    priority: 2,
    instructions: 'Updated instructions',
    options: {
      timestamping: true,
    },
  }),
})

const data = await response.json()
```

## Error Codes

| Status Code | Description                |
| ----------- | -------------------------- |
| 200         | Order Updated Successfully |
| 400         | Invalid Request            |
| 401         | Unauthorized               |
| 403         | Forbidden                  |
| 404         | Order Not Found            |
| 500         | Server Error               |
