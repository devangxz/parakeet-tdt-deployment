# List Orders

Get a list of orders for the authenticated user.

## Endpoint

```http
GET ${baseUrl}/orders
```

## Query Parameters

| Parameter | Type   | Description                             |
| --------- | ------ | --------------------------------------- |
| page      | number | Page number for pagination (default: 1) |
| limit     | number | Number of orders per page (default: 10) |
| status    | string | Filter by order status                  |
| startDate | string | Filter by start date (YYYY-MM-DD)       |
| endDate   | string | Filter by end date (YYYY-MM-DD)         |

## Response

```typescript
interface OrderListResponse {
  success: boolean
  data: {
    orders: {
      id: number
      fileId: string
      status: OrderStatus
      orderTs: string
      deadlineTs: string
      deliveryTs?: string
      priority: number
      tat: number
      instructions?: string
      file: {
        filename: string
        duration: number
        filesize: string
      }
    }[]
    total: number
    page: number
    limit: number
  }
}
```

## Example Request

```typescript
const response = await fetch(
  '${baseUrl}/orders?page=1&limit=20&status=PENDING',
  {
    headers: {
      Authorization: 'Basic YOUR_API_KEY',
    },
  }
)

const data = await response.json()
```

## Error Codes

| Status Code | Description  |
| ----------- | ------------ |
| 200         | Success      |
| 401         | Unauthorized |
| 500         | Server Error |
