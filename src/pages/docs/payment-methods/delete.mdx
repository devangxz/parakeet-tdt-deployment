# Remove Payment Method

Remove a payment method from the user's account.

## Endpoint

```http
DELETE ${baseUrl}/users/{userId}/payment-methods/{paymentMethodId}
```

## Path Parameters

| Parameter       | Type   | Description              |
| --------------- | ------ | ------------------------ |
| userId          | number | ID of the user           |
| paymentMethodId | string | ID of the payment method |

## Response

```typescript
interface RemovePaymentMethodResponse {
  success: boolean
  data: {
    id: string
    deletedAt: string
  }
}
```

## Example Request

```typescript
const response = await fetch('${baseUrl}/users/123/payment-methods/pm_xyz789', {
  method: 'DELETE',
  headers: {
    Authorization: 'Basic YOUR_API_KEY',
  },
})

const data = await response.json()
```

## Error Codes

| Status Code | Description                          |
| ----------- | ------------------------------------ |
| 200         | Payment Method Removed Successfully  |
| 400         | Cannot Remove Default Payment Method |
| 401         | Unauthorized                         |
| 403         | Forbidden                            |
| 404         | Payment Method Not Found             |
| 500         | Server Error                         |
