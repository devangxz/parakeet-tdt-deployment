# List Invoices

Get a list of invoices.

## Endpoint

```http
GET ${baseUrl}/invoices
```

## Response

```typescript
interface InvoiceListResponse {
  success: boolean
  data: {
    u: string
    fullname: string
    email: string
    id: string
    amt: number
    ts: string
    t: string
    ra: number
    cr: number
  }[]
}
```

## Example Request

```typescript
const response = await fetch('${baseUrl}/invoices', {
  headers: {
    Authorization: 'Basic YOUR_API_KEY',
  },
})

const data = await response.json()
```

## Error Codes

| Status Code | Description  |
| ----------- | ------------ |
| 200         | Success      |
| 401         | Unauthorized |
| 403         | Forbidden    |
| 500         | Server Error |
